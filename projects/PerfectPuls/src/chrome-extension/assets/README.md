# Chrome Extension Assets

This folder should contain:
- icon-16.png (16x16 pixels)
- icon-32.png (32x32 pixels)  
- icon-48.png (48x48 pixels)
- icon-128.png (128x128 pixels)

For now, you can:
1. Use any health-related icon (🏥 medical cross, etc.)
2. Create simple PNG files with these dimensions
3. Or Chrome will use default icons

The extension will work without custom icons during development.
